# Ziegenhof-Simulator

Ein einfaches interaktives Klick‑Spiel in Java: Ziegenmilch sammeln, verkaufen und (hoffentlich) Millionär werden.

## Kurzübersicht
Beschreibe hier kurz das Ziel und das Gameplay (1–2 Sätze).


# Jar ausführen (Beispielname)
java -jar target/ziegenhof-simulator-1.0.jar
```

## Gameplay / Steuerung
- Linksklick: Ziege melken
- Buttons: Milch verkaufen, Upgrades kaufen
- Ziel: 1.000.000 Geld in so wenig Tagen(Ticks) wie möglich
